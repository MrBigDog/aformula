Detailed Description

Class which parses a string formula into an abstract syntax tree.

This class keeps track of registering variables and parsing a formula into the various ExprAST classes.

◆ parseString()

ExprAST * AFormula::Private::Parser::parseString ( const std::string &  f)

Parse a complete formula string.

Returns
An ExprAST corresponding to the compelte expression, or NULL on failure.

This function parses a complete formula into an expression tree. This is the primary entry point for the Parser class.

If a NULL value is returned, the function did not parse. A fairly informative error message will be stored in Formula::errorString, suitable for display to the user.
