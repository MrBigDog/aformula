Detailed Description

Namespace containing code private to the AFormula implementation.

No code in the Private namespace should be visible to users of the AFormula library, nor should it be usable.

◆ timerFrequency()

uint64_t AFormula::Private::timerFrequency ( )

Return frequency of high-resolution native timer.

Returns
Frequency of timer, in seconds.

We need a platform-specific, high-resolution timer in order to test which formula-compilation backend performs the fastest at runtime. This function returns that timer's frequency, in Hz. (Execute timerTime() / timerFrequency() to get the elapsed time in seconds.)
